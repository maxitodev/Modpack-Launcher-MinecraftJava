# Coloca aquí tus resource packs (.zip)

Todos los archivos .zip que coloques en esta carpeta serán 
copiados automáticamente a .minecraft/resourcepacks durante la instalación.

Ejemplo de resource packs populares:
- Faithful 32x
- Sphax PureBDcraft
- Soartex Fanver
- Default 3D
- etc.

IMPORTANTE:
- Solo archivos .zip
- Los usuarios podrán activarlos desde Opciones → Resource Packs en Minecraft
