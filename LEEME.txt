───────────────────────
1. Abre el Minecraft Launcher
2. Busca el perfil: "Modpack - by MaxitoDev - Minecraft 1.21.11"
3. Haz clic en JUGAR

  PROBLEMAS COMUNES
═══════════════════════════════════════════════════════════════════

PROBLEMA: "Java no esta instalado"
SOLUCION: Descarga e instala Java desde https://adoptium.net/


PROBLEMA: El instalador no abre
SOLUCION: 
  - Haz clic derecho en Modpack.exe
  - Selecciona "Ejecutar como administrador"
  - Si tu antivirus lo bloquea, permitelo temporalmente


PROBLEMA: No veo el perfil en el launcher
SOLUCION: 
  - Cierra completamente el Minecraft Launcher
  - Abrelo de nuevo
  - El perfil deberia aparecer ahora


PROBLEMA: El juego crashea al iniciar
SOLUCION:
  - Verifica que tengas Java 17 o superior
  - Asegurate de tener suficiente RAM asignada (minimo 8GB)


PROBLEMA: El juego va lento o con lag
SOLUCION: Aumenta la RAM asignada al perfil
  1. Abre el Minecraft Launcher
  2. Selecciona el perfil del modpack
  3. Haz clic en los tres puntos (...) → Editar
  4. Ve a "Mas opciones" o "More Options"
  5. En "Argumentos JVM" busca "-Xmx8G"
  6. Cambia el numero segun tu RAM:
     - Si tienes 16GB RAM → usa -Xmx10G o -Xmx12G
     - Si tienes 12GB RAM → usa -Xmx8G
     - Si tienes 8GB RAM → deja -Xmx6G
  7. Guarda y prueba de nuevo

═══════════════════════════════════════════════════════════════════
  CONFIGURACION POST-INSTALACION
═══════════════════════════════════════════════════════════════════

ACTIVAR RESOURCE PACKS:
──────────────────────
1. Inicia Minecraft con el perfil del modpack
2. Ve a Opciones → Resource Packs (Paquetes de Recursos)
3. Activa los siguientes packs (muevelos a la derecha):
   - Alacrity
   - FreshAnimations
4. Haz clic en "Done" (Hecho)
5. El juego se recargara automaticamente

CAMBIAR SHADERS:
───────────────
1. En el juego, presiona ESC
2. Ve a Opciones → Video Settings → Shaders
3. Selecciona el shader que prefieras:
   - Bliss
   - Complementary Reimagined
   - Complementary Unbound
