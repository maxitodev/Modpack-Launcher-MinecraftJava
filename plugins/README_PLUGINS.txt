# PLUGINS DEL SERVIDOR

Esta carpeta contiene los plugins que se instalarán en el servidor.

## Importante
- Los plugins son para servidores Bukkit/Spigot/Paper
- Los archivos .jar de los plugins van directamente en esta carpeta
- El servidor vanilla de Minecraft NO soporta plugins

## Instalación
Los plugins de esta carpeta se copiarán automáticamente al servidor
durante el proceso de build.

## Estructura
plugins/
  ├── plugin1.jar
  ├── plugin2.jar
  └── ...

## Nota
Si usas servidor vanilla, esta carpeta no tendrá efecto.
Para usar plugins necesitas Paper/Spigot/Bukkit.
