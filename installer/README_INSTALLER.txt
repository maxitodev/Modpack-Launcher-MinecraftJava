# Coloca aquí el instalador de NeoForge

Descarga el instalador de NeoForge desde:
https://neoforged.net/

IMPORTANTE:
- Solo debe haber UN archivo .jar en esta carpeta
- El nombre debe comenzar con "neoforge-"
- Ejemplo: neoforge-1.20.1-47.1.3-installer.jar

VERSIONES RECOMENDADAS:
- Para Minecraft 1.20.1: NeoForge 47.1.3 o superior
- Para Minecraft 1.20.4: NeoForge 20.4.x
- Para Minecraft 1.21: NeoForge 21.0.x

Asegúrate de que todos tus mods sean compatibles con la 
versión de NeoForge que elijas.
